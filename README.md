## Connecting the M5Stack Core2 to the Pololu 3Pi+

Please double check your connections!  The devices must have their grounds (GND) connected.

| Arduino Pin | Core2 Pin |
|-------------|-----------|
| GND         | GND       |
| 2 (SDA)     | G32 (SDA) |
| 3 (SCL)     | G33 (SCL) |
